# Python_Projects

A personal **Python monorepo**: a collection of projects, experiments, and learning work across **Data Engineering**, **Machine Learning**, **MLOps**, and **RAG (Retrieval-Augmented Generation)**.


---

## Repository Structure

Top-level folders:

- `Data_Engineering_Project/` — data-engineering related work (pipelines, ETL-style scripts, data handling, etc.)
- `Intern_Intelligence_Projects/` — internship/assignment-style projects and exercises
- `ML_Projects/` — machine learning experiments and notebooks
- `Ml_ops_project/` — MLOps-focused work (packaging, deployment patterns, automation, etc.)
- `Rag_ml_ops/` — RAG + MLOps experiments (LLM tooling, retrieval, orchestration, etc.)
- `Python/` — general Python scripts, utilities, and smaller projects
